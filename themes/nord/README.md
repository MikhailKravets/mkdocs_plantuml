Nord

-----------------

[Nord](https://www.nordtheme.com) is a cold, minimalist color palette that was applied to
for PlantUML.

## How to use

In order to use this theme with `mkdocs_puml`, set `theme` config of the plugin as follows:

```yml
theme:
    light: nord/light
    dark: nord/dark
```

## Flavors

This theme comes in two flavors

- `day` for light mode
- `night` for dark mode
