    Kanagawa ¶

    Kanagawa theme implements a color palette defined in kanagawa.nvim. This color scheme is inspired by Katsushika Hokusaiâ€™s work, especially, his famous "The Great Wave".

    How to use¶

    In order to use this theme with mkdocs_puml, set theme config of the plugin as follows:

    plantuml:
  theme:
    light: kanagawa/fuji
    dark: kanagawa/wave

    Flavors¶

    This theme has two flavors

    • fuji for light mode
    • wave for dark mode
